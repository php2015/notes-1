

#### 方法1 cdn

1. 引入

```html
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Acme" type="text/css">
```

2. 设置font-family即可

```css
div { font-family: Acme; }
```





#### 方法2 下载字体使用

1. 下载  https://fonts.google.com/

2. 引入

```css
@font-face {
  font-family: "Acme";
  src: url(./Acme/Acme-Regular.ttf);
}
```

3. 设置font-family

```css
div { font-family: Acme; }
```
