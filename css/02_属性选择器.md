

#### css2

较为严格，需要是完整单词

```css
[attr]
[attr1][attr2]
[attr=val]
[attr~=val]
[attr|=val]
```

##### 

##### 理解[attr~=special]

class中有一个类名为special即可，如"special normal"，注意"special-cat"不行



##### 理解[attr|=en]

class只有en符合

或者en-开头，如en-us，这种情况有其他类名也可以





#### css3

```css
[attr*=val]  包含val，位置不限有即可
[attr^=val]  开头是val即可
[attr$=val]  结尾是val即可
```
