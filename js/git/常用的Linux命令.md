## 常用的Linux命令

```
cd ..                       - 回到上级目录
cd filename                 - 进入当前目录
pwd                         - 显示当前所在目录
ls                          - 列出当前目录的所有文件
clear                       - 清屏
reset                       - 跟clear效果一样 重新初始化终端
touch kimi.js               - 新建kimi.js文件
rm kimi.js                  - 删除kimi.js文件
mkdir kimi                  - 创建kimi文件夹
rm -r kimi                  - 删除kimi文件夹
mv kimi.js kimi             - 把kimi.js文件移动到kimi文件夹
history                     - 查看历史命令
help                        - 帮助
exit                        - 退出
```
