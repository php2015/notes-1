#### 安装

1. 安装node v8.9以上

2. 全局安装webpack

   ```js
   npm i webpack -g
   ```

3. 全局安装脚手架即可，不用本地安装

   ```js
   // vue-cli 3.x
   npm i @vue/cli -g
   
   // 拉取vue-cli 2.x模板
   npm install -g @vue/cli-init@3.2.0
   ```

4. 创建项目

   ```js
   // 3.x
   vue create project
   
   // 2.x
   vue init webpack project
   ```
